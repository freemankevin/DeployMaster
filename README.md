# DeployMaster

🚀 **DeployMaster** 是一个现代化的自动化部署平台，提供 SSH 主机管理、终端连接、密钥管理等功能。


## 开发命令

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview

# 初始化数据库
npm run init-db
```

## 环境配置

### 后端环境变量

创建 `backend/.env` 文件：

```env
DATABASE_URL=sqlite:///./data/deploymaster.db
SECRET_KEY=your-secret-key
DEBUG=false
```
